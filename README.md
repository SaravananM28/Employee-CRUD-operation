👨‍💼 Employee CRUD Operation (PHP & MySQL)

A simple and efficient "Employee Management System built using Core PHP, MySQL, HTML, CSS, and Bootstrap".  
This project demonstrates basic "CRUD operations (Create, Read, Update, Delete)** for managing employee data".

---

## 🚀 Features

- ➕ Add new employee
- 📋 View all employees
- ✏️ Edit employee details
- ❌ Delete employee
- 🔄 Dynamic data loading (AJAX-based)
- 💻 Responsive UI using Bootstrap

---

## 🛠️ Technologies Used

### Frontend:
- HTML5  
- CSS3  
- Bootstrap  
- JavaScript / AJAX  

### Backend:
- Core PHP  

### Database:
- MySQL
